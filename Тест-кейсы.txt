Автоматизированные тест-кейсы


# Login Page

1. Авторизация поьзователя с валидными данными.
  1. Открыть страницу https://www.saucedemo.com/ | Старница авторизации открыта
  2. Ввести имя пользователя (standard_user)     | Имя пользователя введено
  3. Ввести пароль (secret_sauce)                | Пароль введен
  4. Нажать кнопку логин                         | Логин успешен, открыта страница продуктов


2. Авторизации заблокированного пользователя.
  1. Открыть страницу https://www.saucedemo.com/ | Старница авторизации открыта
  2. Ввести имя пользователя (locked_out_user)   | Имя пользователя введено
  3. Ввести пароль (secret_sauce)                | Пароль введен
  4. Нажать кнопку логин                         | Ошибка авторизации. Страница продуктов не открыта.
					         | Сообщение об ошибке содержит текст: "Epic sadface: Sorry, this user has been locked out."


3. Авторизация пользователя с неверными данными
  Данные:
  1. userName = "abc", password= "", message = "Epic sadface: Password is required"
  2. userName = "", password= "abc", message = "Epic sadface: Username is required"
  3. userName = "abc", password= "abc", message = "Epic sadface: Username and password do not match any user in this service"
  4. userName = "", password= "", message = "Epic sadface: Username is required"
  
  Шаги:
  1. Открыть страницу https://www.saucedemo.com/ | Старница авторизации открыта
  2. Ввести имя пользователя (userName)          | Имя пользователя введено
  3. Ввести пароль (password)                    | Пароль введен
  4. Нажать кнопку логин                         | Ошибка авторизации. Страница продуктов не открыта.
					         | Сообщение об ошибке содержит текст: message
